# Multilingual Intent Classification

## Use Case
Classify user intents across multiple languages using semantic embeddings and K-Nearest Neighbors, enabling single-model training for multilingual customer service systems.

## Modalities
- Text (multiple languages)

## AWS Service Integration
- Amazon Bedrock (Nova Multimodal Embedding model for cross-lingual semantic representations)
- scikit-learn (K-Nearest Neighbors classifier for intent prediction)
