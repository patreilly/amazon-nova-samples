# Visual Product Search with Image-Text Embeddings

## Use Case
Enable customers to search for products using images or text descriptions, finding visually similar items in e-commerce catalogs.

## Modalities
- Images
- Text

## AWS Service Integration
- Amazon Bedrock (Nova Multimodal Embedding model for image and text embeddings)
- Amazon S3 (product catalog storage)
- Amazon S3 Vectors (vector db)
- Amazon Open Search Severless vector store (vector db)
